C语言构成，原理，基础语法等等

构成

c语言主要由以下几个方面构成

  • 预处理指令：#include #define 作为再编译前的预处理

  • 主函数：每个c语言只能有一个main函数，作为执行程序的位置

  • 函数定义：定义使用的函数，描述函数的运行逻辑

  • 变量声明：所以变量声明前都需要使用指令声明，要求必须在代码开头

    实际演示

原理

程序的c代码先预处理编译成汇编语言，将汇编语言转换为机器码二进制指令输出.o文件，最后链接合并成可执行的.exe文件
