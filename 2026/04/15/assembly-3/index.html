x86 汇编进阶笔记：条件判断、循环、函数与段定义

一、条件判断指令

1. CMP 指令（Compare）

指令 作用 对标志位的影响
cmp dest, src 计算 dest - src， 不保存结果 ，仅影响标志位 ZF, CF, OF, SF, AF, PF

assembly

2. 条件跳转指令（Conditional Jump）

指令 全称 跳转条件 适用场景
JE Jump if Equal ZF = 1 相等则跳
JZ Jump if Zero ZF = 1 结果为零则跳
JNE Jump if Not Equal ZF = 0 不相等则跳
JNZ Jump if Not Zero ZF = 0 结果非零则跳

无符号数比较：

指令 全称 跳转条件 含义
JA Jump if Above CF=0 且 ZF=0 大于则跳（无符号）
JNBE Jump if Not Below or Equal 同上 同上
JB Jump if Below CF=1 小于则跳（无符号）
JNAE Jump if Not Above or Equal 同上 同上
JAE Jump if Above or Equal CF=0 大于等于则跳
JBE Jump if Below or Equal CF=1 或 ZF=1 小于等于则跳

有符号数比较：

指令 全称 跳转条件
JG Jump if Greater SF=OF 且 ZF=0
JL Jump if Less SF≠OF
JGE Jump if Greater or Equal SF=OF
JLE Jump if Less or Equal SF≠OF 或 ZF=1

4. ADC 指令（Add with Carry）

指令 作用 公式
adc dest, src 带进位加法 dest = dest + src + CF

应用场景：大数加法（超过寄存器位宽）

5. SBB 指令（Subtract with Borrow）

指令 作用 公式
sbb dest, src 带借位减法 dest = dest - src - CF

二、循环指令 LOOP

1. LOOP 指令详解

指令 操作流程 CX/ECX/RCX 变化
loop label 1. CX = CX - 1``2. 若 CX ≠ 0，跳转到 label 自动递减

注意 ：loop 默认使用 CX （16位）、 ECX （32位）或 RCX （64位）作为计数器。

2. 循环相关指令

指令 全称 跳转条件
loop label Loop CX ≠ 0 时跳转
loope label Loop while Equal CX ≠ 0且 ZF=1 时跳转
loopz label Loop while Zero 同上
loopne label Loop while Not Equal CX ≠ 0且 ZF=0 时跳转
loopnz label Loop while Not Zero 同上

三、函数调用指令 CALL 与 RET

1. CALL 指令

指令格式 操作 本质
call near label 1. 将下一条指令地址（IP）压栈``2. IP = label 的偏移地址 段内调用
call far ptr label 1. 将 CS 压栈2. 将 IP 压栈3. CS:IP = 新的段:偏移 段间调用
call reg 将 IP 压栈，IP = 寄存器值 间接调用
call [mem] 将 IP 压栈，IP = 内存值 内存间接调用

2. RET 指令

指令格式 操作 本质
ret 1. 从栈顶弹出 IP``2. 跳转到该地址执行 段内返回
ret far 1. 从栈顶弹出 IP2. 从栈顶弹出 CS3. 跳转到 CS:IP 段间返回
ret n 1. 弹出 IP``2. SP = SP + n（清理参数） 带参数清理的返回
