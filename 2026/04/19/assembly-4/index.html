GDB调试学习笔记（含汇编调试）

什么是GDB

GDB（GNU Debugger）是Linux下的C/C++调试工具，可以让你一步步运行程序，查看变量值，查看汇编指令，找出bug。

准备工作：编译时加调试信息

启动GDB

核心命令速查表

命令 简写 作用
break b 设置断点
run r 运行程序
continue c 继续运行到下一个断点
next n 单步执行（不进入函数）
step s 单步执行（进入函数）
print p 打印变量值
list l 查看源代码
backtrace bt 查看函数调用栈
info i 查看信息
quit q 退出GDB

汇编调试相关命令

命令 简写 作用
disassemble disas 查看汇编代码
stepi si 单步执行一条汇编指令
nexti ni 单步执行（不进入函数调用）
info registers i r 查看寄存器值
print $rax p $rax 打印寄存器值
x x 查看内存
layout asm - 显示汇编布局
layout reg - 显示寄存器布局

寄存器 作用
